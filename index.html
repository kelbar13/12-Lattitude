Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. Using the OpenWeatherMap API, we collected data from over 500 cities around the world.

We established a dataset, then used Matplotlib to plot four different characteristics of weather vs.latitude. These characteristics include temperature, cloudiness, wind speed, and humidity. In this site you will find the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
